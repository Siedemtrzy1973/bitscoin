How to mine BitsCoin?

  1. Download the wallet:
    • BitsCoin-Qt.AppImage (Linux)
    • BitsCoin-Qt ZIP (Windows)
  2. Generate a BSC address in the wallet (Menu ‚Üí Receive). Copy the generated address.
  3. Download mining software:
    • cpuminer-opt
    • cgminer
  4. Start mining with the following parameters: 
    cgminer -o stratum+tcp://pool.bitscoin.online:3333 -u YOUR_BSC_ADDRESS -p x

    Or for cpuminer:

    cpuminer-avx2 -a sha256d -o stratum+tcp://pool.bitscoin.online:3333 -u YOUR_BSC_ADDRESS -p x -t THREADS
    • -a sha256d ‚Üí algorithm SHA-256
    • -o ‚Üí pool address (stratum)
    • -u YOUR_BSC_ADDRESS ‚Üí Your BSC address from wallet
    • -p x ‚Üí password (usually x)
    • -t THREADS ‚Üí number of CPU threads (for cpuminer)

GPG Signature Verification

For security, we recommend verifying the GPG signature of the downloaded BitsCoin wallet.

1. Download the wallet and signature:

wget https://bitscoin.online/download/BitsCoin-Qt.AppImage
wget https://bitscoin.online/download/BitsCoin-Qt.AppImage.asc

or for Windows version:

wget https://bitscoin.online/download/BitsCoin-Qt-Win64.zip
wget https://bitscoin.online/download/BitsCoin-Qt-Win64.zip.asc

2. Import the BitsCoin public key (if you haven't already):

gpg --keyserver keyserver.ubuntu.com --recv-keys YOUR_KEY_ID

3. Verify the signature:

gpg --verify BitsCoin-Qt.AppImage.asc BitsCoin-Qt.AppImage

or for Windows ZIP:

gpg --verify BitsCoin-Qt-Win64.zip.asc BitsCoin-Qt-Win64.zip

A valid signature means the file comes from the official BitsCoin team and has not been modified.
